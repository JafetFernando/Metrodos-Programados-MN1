# Metrodos-Programados-MN1
Programa para visualizar la resolucion de metodos numericos: Método de Bisección, Método de Newton-Raphson, Método de la Secante, Ecuaciones lineales: Eliminación de Gauss, Método de Jacobi, Método de Gauss-Seidel, Factorización LU: Método de Crout, Método de Cholesky, Valores y vectores propios: Método de la Potencia, Método de la Potencia Inversa
